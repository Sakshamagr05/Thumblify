# 🚀 Thumblify - AI Powered Thumbnail Generator

Thumblify is a full-stack AI-powered thumbnail generation platform that helps creators generate attractive and professional thumbnails using Artificial Intelligence.

The platform combines a modern animated user interface with AI image generation capabilities to create high-quality thumbnails from text prompts.
